# Mechanism

- Hacker creates a malicious link that contains a script to perform vandal activities to the victim such as stealing cookies, login credentials, private data, etc.
- Hacker sends the link to the victim.
- When the victim clicks on the link, the victim will be redirected to the attacked site with the malicious script of the hacker and the victim's information will be exposed to the hacker
